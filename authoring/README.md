# Authoring Workspace

Thư mục này dành cho việc biên soạn nội dung mới, không phụ thuộc vào việc có tài liệu gốc để dịch hay không.

Nên dùng thư mục này cho:

- viết course brief;
- dựng module outline;
- soạn session plan;
- thiết kế activity và assessment;
- chuẩn bị note cho giảng viên hoặc tài liệu handout.

## Templates

Các template mẫu nằm trong `authoring/templates/`:

- `course-brief.md`
- `module-outline.md`
- `session-plan.md`
- `activity-sheet.md`
- `assessment-blueprint.md`

## Cách dùng

Có hai cách:

1. Soạn trực tiếp trong `authoring/` khi đang xây framework chung.
2. Sao chép template sang `courses/<code>/03_authoring/` khi làm cho một khóa cụ thể.
